# Box 3 zonder gedoe (static site)

Deze map bevat een iPhone‑first landing page met:
- scroll‑progress bar bovenin
- scroll reveal animaties
- geen dependencies (pure HTML/CSS/JS)

## Live zetten op GitHub Pages
1. Maak een repo op GitHub (bijv. `box3-zonder-gedoe`)
2. Upload alle bestanden uit deze map (in de root van je repo)
3. Ga naar **Settings → Pages**
4. Kies: **Deploy from a branch**
5. Branch: **main**, Folder: **/ (root)**
6. Save → je site komt op `https://<username>.github.io/<repo>/`

## Structuur
- `index.html`
- `assets/css/styles.css`
- `assets/js/app.js`
- `assets/img/*` (favicon, icons, og image)
